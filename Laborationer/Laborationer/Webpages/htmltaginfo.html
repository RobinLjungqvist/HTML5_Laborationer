Header

Keyword header within opening and closing tags. < >. The header is meant to contain a logo, or a headline for the website. It also usually contains navigation.

Navigation

Keyword nav within opening and closing tags < >. The navigation is a list of link to the other pages of the website.

Section

Keyword section within opening and closing tags < >. The section is used to grouping distinct content or functionality.

Article

Keyword article within opening and closing tags < >. Is related to section but articles contains related content that can standalone without the rest of the website.

Aside

Keyword aside within opening and closing tags < >. Aside is supposed to contain limited information related to an article but should be the article. Commonly used for Ads and related content.

Footer

Keyword footer within opening and closing tags < >. Footer is connected to an article or the whole website and should contain contact information who wrote the website and legal notation such as copyright.

abbr & blockquote

Abbr stand for abbreviation or an acronym, like NASA. It is used for screenreaders to read out the meaning of an acronym.

Blockquote

The blockquote tag specifies a section that is quoted from another source. Browsers usually indent blockquote elements.

figcaption

del & ins & s tags

When i want to show that something has been removed i use the del tag. When i want to show that something has been added i use the ins tag. An old way of making a line through a word was the s tag, but if you want to show that text has been removed or replaced use the del or ins tag instead.

table & col & colgroup, tr and th

Table defines a table with column rows, header cells and data cells. Colgroup tag defines a collection of cells in a table for formatting. the col tag defines the properties of the selected columns.

tr is table row th is table header cell
td is for table data cell And you can write stuff in cells.
