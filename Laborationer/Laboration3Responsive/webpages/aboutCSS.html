Colors

The CSS attribute color defines the textcolor of that element. It can be defined as RGBA, HEX, HSLA and predefined names.

The different colors

I will write the word color 4 times with the different color definition values and different background colors. They are defined by writing color: (type) where type can be rgb, hex or hsl or the predefined name of a color. ex. hotpink.

More on HSLA

It is similar to RGBa in that you declare three values determining the color and then a fourth value for its transparency level. You can read more about browser support below, but it's basically any browser that supports rgba supports hsla too.

  • Hue - Think of a color wheel. Around 0o and 360o are reds 120o are greens, 240o are blues. Use anything in between 0-360. Values above and below will be modulus 360.
  • Saturation - 0% is completely denatured (grayscale). 100% is fully saturated (full color).
  • Lightness - 0% is completely dark (black). 100% is completely light (white). 50% is average lightness.
  • Alpha - Opacity/Transparency value. 0 is fully transparent. 1 is fully opaque. 0.5 is 50% transparent.

Contrast

Contrast are the concept of having a certain color on a different background. Which can make the contrast low, high or medium. Text is harder to read when the contrast is low. High contrast makes the text easier to read but too much can also make the text hard to read. That's why you should do as the swedes, take the middle route. Go medium!

Opacity

Colors have the ability to be more or less see-through which can come in handy. This is called Opacity or alpha. Some colors value definitions have this built in but you can also specify it on its own with the keyword opacity with a value between 0.0 and 1.0.

Typeface

There are a few diffrent font types which are called serif, sans-serif and monospace. Serif have extra details where as sans-serif has none. monospace means that every letter has a fixed width.

Type styles

There are three typical styles that you can format your font with. These are Weight, style and stretch. They are implemented by typing the keyword followed by the value. font-weight:(light, medium, bold..) font-style: (normal, italic oblique) or font-stretch: (condensed, extended) Below in the example i will write serif as bold, sans-serif as italic and monospace as condensed.

@font-face

The @fontface keyword lets you use a font that isn't installed on your computer by getting it from an url!

Example: @font-face{ font-family: 'MyOwnFont'; src: url('fonts/MyOwnFont.eot'); }
